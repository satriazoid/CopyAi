# 🚀 AI Text Cleaner

Aplikasi Python untuk membersihkan teks hasil AI agar lebih rapi, natural, dan siap digunakan.

Dibuat dengan 2 versi UI:
- **PyQt6 (Modern UI)**
- **ttkbootstrap (Tkinter version)**


## ✨ Features

-  Remove excessive spaces  
-  Clean AI formatting (bullet, symbol, dll)  
-  Replace dash (`- – —`) menjadi spasi  
-  Normalize paragraph & line break  
-  Copy hasil dengan 1 klik  
-  Dark / Light mode (PyQt version)  
-  UI modern & responsive  


## Preview

### PyQt Version (Recommended)
- Modern UI  
- Rounded button  
- Theme switch (dark/light)  

### ttkbootstrap Version
- Simpler UI  
- Lightweight  
- Cocok untuk basic usage
